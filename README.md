## Bank Management System

This project is a banking management system with different interfaces for users, tellers, and administrators. It provides functionalities such as account management, transaction handling, loan processing, card management, and locker management.

### User Interface
- **Transaction Tracking**: Users can view their transaction history and account statements.
- **Apply for Card**: Users can apply for a new card by providing necessary details.

### Teller Interface
- **Transaction Processing**: Perform deposit and withdrawal transactions on behalf of users.
- **New Account Creation**: Create new accounts for customers.
- **Notification Management**: Send notifications to users regarding account activities.
- **Locker Management**: Assign and manage lockers for customers.
- **Loan Processing**: Handle loan applications and approvals for customers.

### Admin Interface
- **Employee Management**: Admins can manage employee information and salaries.
- **Branch Management**: Admins can oversee branch operations, including branch-specific tasks.
- **Interest Rate Management**: Admins can set and adjust interest rates on savings accounts.

## Setup
1. Clone the repository.
2. Download the SQL files for creating the database tables.
3. Configure database connection settings.
4. Run the SQL files to create the necessary tables.
5. Download the frontend files into one directory.
6. Run the file `LoginFrame.java` to start the application.

## Usage
- **User Interface:**
  - Users can log in using their credentials and perform various banking operations, including viewing transaction history, account statements, and applying for a new card.

- **Teller Interface:**
  - Tellers can log in using their credentials to serve customers and manage bank operations. They can process transactions, create new accounts, manage lockers, and handle loan applications.

- **Admin Interface:**
  - Admins can log in using their credentials to configure system settings and manage bank operations. They can manage employees, branches, interest rates, and oversee overall bank operations.

## Technologies Used
- **Database:** SQL
- **Frontend:** Java Swing
